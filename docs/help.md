# Help

## Keyboard shortcuts

Navigate the site without touching the mouse. Press <kbd>g</kbd> followed by a letter
within one second:

| Shortcut                       | Action                       |
| ------------------------------ | ---------------------------- |
| <kbd>g</kbd> then <kbd>h</kbd> | Go to the home page          |
| <kbd>g</kbd> then <kbd>r</kbd> | Browse all rules             |
| <kbd>g</kbd> then <kbd>s</kbd> | Go to sources                |
| <kbd>g</kbd> then <kbd>c</kbd> | Go to the catalogue overview |
| <kbd>g</kbd> then <kbd>l</kbd> | Go to the changelog          |
| <kbd>?</kbd>                   | Open this page               |

After navigating, focus lands in the page's search box where one is available.

The built-in site search opens with <kbd>Ctrl</kbd>+<kbd>K</kbd> (or
<kbd>⌘</kbd>+<kbd>K</kbd> on macOS).
