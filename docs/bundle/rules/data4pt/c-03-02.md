# Notes

Depends on Frame type. - For all frames check and DefaultLanguage exists in DefaultLocale in FrameDefaults. - For frames that contain spatial coordinates, check that default LOcationSystem is specified (usually WGS84) - For Frames that contain elements with Timezones. (e.g. STOP PLACEs etc. in . SITE FRAME. SCHEDULED STOP POINT in SERVICE FRAME, Check that Time Zone is specified. - For frames that hold monetary values, e.g. FARE FRAMES or if amount specified . NB can be specified on outmost COMPOSITE FRAME if common to all.
