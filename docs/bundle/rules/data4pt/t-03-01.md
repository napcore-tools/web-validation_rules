# Notes

Primary SERVICE JOURNEYs making up the timetable journeys should each follow a SERVICE PATTERN (secondary journeys, e.g. for interchanges etc. do not need to specify details) . Also if using CALLs not essential. Upstream numbering clash: T.03.01 is assigned to 3 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
