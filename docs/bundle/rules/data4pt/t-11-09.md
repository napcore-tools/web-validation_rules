# Notes

The StartTime. EndTime. FromPointRef and ToPointRef elements of JOURNEY PART COUPLE must be consistent with the joined JOURNEYPARTs and theMainPartRef of the JOURNEY PARTs that it references (must find the same values on the parts and in the JOURNEY PART COUPLE)

# Details

- **Result:** Error
- **Scope:** **area:** Timetable, **aspect:** Journey Parts, **applies_to:** All usages
- **Mechanism:** code
