# Notes

If a SCHEDULED STOP POINT is assigned to a STOP PLACE and/or QUAY the type of stop should be consistent, e.g. both rail stations, rail platform etc.

# Details

- **Result:** Warning
- **Scope:** **area:** Stop, **aspect:** Stop Point, **applies_to:** All usages
- **Mechanism:** code
