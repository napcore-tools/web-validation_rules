# Notes

QUAYs should not repeat the name of the STOP PLACE but provide a qualifier, e.g. 'Platform 5' that can be if necessary be used in conjunction with that of the name of the STOP PLACE.

# Details

- **Result:** Warning
- **Scope:** **area:** Stop, **aspect:** Hierarchy, **applies_to:** EPIP
- **Mechanism:** code
