# Notes

The JourneyDuration attribute of SERVICE JOURNEY should contain information of approximately the same value as the difference between starting at the first stop and arriving at the last. A tolerance for rounding is acceptable (5 or 10 minutes maximum)

# Details

- **Result:** Warning
- **Scope:** **area:** Timetable, **aspect:** Timings, **applies_to:** All usages
- **Mechanism:** code
