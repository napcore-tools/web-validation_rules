# Notes

All SITE COMPONENTS (and objects that inherit from them) have a SiteRef attribute, which should reference an object consistent with its nature: - For a QUAY. this is the identifier of the monomodal STOP PLACE on which the QUAY depends. - For an ACCESS this is the identifier of the monomodal STOP PLACE. or multimodal L STOP PLACE to which the ACCESS leads.

# Details

- **Result:** Warning
- **Scope:** **area:** Stop, **aspect:** Stop place, **applies_to:** All usages
- **Mechanism:** code
