# Notes

In the Transfer object and its specializations. we must have the information WalkTransferDuration.DefaultDuration. Note that values may be defaulted from SITE CONNECTION and/or DEFAULT CONNECTION. Upstream numbering clash: T.21.01 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
