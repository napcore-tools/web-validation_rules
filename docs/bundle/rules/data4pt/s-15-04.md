# Notes

For TOPOGRAPHIC LOCATIONS. the ParentTopographicPlaceRef attribute respects the following rules: - a Region has no parent (but necessarily a CountryRef) - a Department is contained in a Region - a Grouping of Municipalities is contained in a Department (or possibly a region if it straddles several DEPARTMENTS) - a City is contained in a Department (and NOT in Grouping Of Communes: see containedIn below) - a Suburb is contained in City - a Village / parish/ hamlet is contained in a Department

# Details

- **Result:** Warning
- **Scope:** **area:** Stop, **aspect:** Topographic, **applies_to:** All usages
- **Mechanism:** code
