# Notes

If the Journey has JOURNEY PARTs. the parts must be consistent with the SERVICE JOURNEY. i.e. use a subset of the SCHEDULED STOP POINTs and in the same running order.

# Details

- **Result:** Error
- **Scope:** **area:** Timetable, **aspect:** Journey Parts, **applies_to:** All usages
- **Mechanism:** code
