# Notes

If a validity is specified on a SERVICE JOURNEY it must be in between the FromDate and ToDate of the containing VERSION FRAME.

- **Result:** Error
- **Scope:** **area:** Timetable, **aspect:** Dates, **applies_to:** All usages
