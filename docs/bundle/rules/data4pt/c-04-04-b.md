# Notes

In the case of simple elements such as NOTICEs. etc. there should be a single instance that is referenced via a separate NOTICE ASSIGNMENT. rather than multiple separate instances that repeat the same content. E.g. if there is a 'Runs on Sunday' notice it should be reused by a reference from each journey to which it applies. not redeclared as a separate entity Code normalized from 'c.04.04' in the source sheet. Upstream numbering clash: C.04.04 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
