# Notes

Modes should be consistent within a document. e.g. if it is a rail timetable. 'rail 'can be the default mode on the frame. and then should be the mode on the stops . VEHICLE JOURNEYs. LINES, etc. Should also be 'rail' i the header topics. Transport submode should be one of the submodes for rail.
