# Notes

If the FLEXIBLE SERVICE PROPERTies of the service indicate a FlexibleServiceType = fixedPassingTimes then the service must be a TEMPLATE SERVICE JOURNEY with a TemplateVehicleJourneyType set to 'Headway'. As a corollary. if FlexibleServiceType is dynamicPassingTimes or fixedPassingTimes: then the interval is not at a regular frequency (and therefore not of type FlexibleServiceTyoe = fixedPassingTimes)

# Details

- **Result:** Error
- **Scope:** **area:** Timetable, **aspect:** Flexible services, **applies_to:** All usages
- **Mechanism:** code
