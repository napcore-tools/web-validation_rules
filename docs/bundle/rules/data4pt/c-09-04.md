# Notes

If a date range for validity is specified on a detailed element it must be in between From Date and To Date of the containing VERSION FRAME.

# Details

- **Result:** Error
- **Scope:** **area:** Common Content, **aspect:** Dates, **applies_to:** All usages
- **Mechanism:** code
