# Notes

STOP PLACES can be organised into hierarchies but certain rules apply to how QUAYs etc. should be organised. - A child STOP PLACE that is a terminal leaf in the hierarchy should normally only contain QUAYs of one mode, i.e. be 'monomodal' . - A 'multimodal' Parent STOP PLACE should only contain other STOP PLACEs,
