# Notes

If a given timetable contains SERVICE JORUNEYS in both directions. both directions of a journey are given. then the reverse calling pattern should be similar to the forward calling pattern. I.e. they lie along a similar line. use SCHEDULED STOP POINTs associated with the same stop places in reverse sequence. etc. Upstream numbering clash: T.02.05 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
