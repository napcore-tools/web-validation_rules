# Notes

XML validator will do this provided a uniqueness constraint has been declared correctky in the schema. - In most cases ID is unique to type of element. - In a few cases id is common to different specialisations of the same type. rule is declared. For example all SiteElements - For some child elements it is unique within id/order rather than id.

# Details

- **Result:** Error
