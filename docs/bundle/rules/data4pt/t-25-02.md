# Notes

If the PointStandingForAZone attribute of FLEXIBLE SERVICE PROPERTies is true. then there must be an associated Zone whose Centroid references this point. Upstream numbering clash: T.25.02 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
