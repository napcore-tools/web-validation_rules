# Notes

Distance between QUAY and STOP PLACE should not be too far apart. STOP PLACE location is centroid of station. QUAY is centroid of QUAY .

# Details

- **Result:** Warning
- **Scope:** **area:** Stop, **aspect:** Spatial, **applies_to:** All usages
- **Mechanism:** code

# Parameters

- `stop_place_quay_distance_by_mode` - Structure with distances in meters for each type of mode for the STOP PLACE (e.g., train station, bus stop, airport, …)
