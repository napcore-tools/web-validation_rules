# Notes

If the Journey has JOURNEY PARTs shared with other journeys, then the JOURNEY PARTs must be separately defined at the root of the XML document and referenced by JourneyPartRefs (and not included inline within a specific SERVICE JOURNEY. If the JOURNEY PARTs are to break up a single journey then they may be inline.
