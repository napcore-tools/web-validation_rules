# Notes

LINE presentation colours for a given LINE should be distinct from the colours used for other LINEs. Background/ font colours should be distinct. . Also Check colour contrast of LINE presentation attributes/ e/g/ black font on black background cannot be read. Upstream numbering clash: T.02.11 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
