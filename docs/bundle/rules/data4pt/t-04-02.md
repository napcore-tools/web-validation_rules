# Notes

GROUPS of SERVICE should be provided that group the SERVICE JOURNEYs into useful groups for human readability. eg 'outbound monday to friday'. 'inbound. monday to friday'. etc. - The GROUP OF SERVICE should have a DIRECTION and DAY TYPE specified. - Outbound and inbound journeys should not be included in the same GROUP OF SERVICES. - Journeys in a GROUP of SERVCIES should normally have a similar DAY TYPE. Code normalized from 'T04.02' in the source sheet.
