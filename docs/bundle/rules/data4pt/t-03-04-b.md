# Notes

A SERVICE PATTERNs comprises a lists of STOP POINTS IN JOURNEY PATTERN. Each of these must reference as SCHEDULED STOP POINT. Upstream numbering clash: T.03.04 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
