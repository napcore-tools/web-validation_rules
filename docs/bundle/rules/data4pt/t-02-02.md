# Notes

Check if TransportSubMode and TransportMode is used for a Line. Then validate that the used SubMode actually is valid for that TransportMode. E.g. localBus is a Bus Upstream numbering clash: T.02.02 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
