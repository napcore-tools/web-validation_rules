# Notes

All frames should have a default CODE SPACE declared. The CODE SPACE declares a default prefix to assume for all identifiers that do not have an explicit prefix. Upstream numbering clash: C.02.05 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
