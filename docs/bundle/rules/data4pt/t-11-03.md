# Notes

The StartTime and EndTime times of a JOURNEY PART. must be between the first and last stop times of the SERVICE JOURNEY . Moreover. if the FromStopPointRef and ToStopPointRef are set. then these times must correspond to an arrival or departure time of the JORUNEY at the corresponding SCHEDULED STOP POINTs.

# Details

- **Result:** Error
- **Scope:** **area:** Timetable, **aspect:** Journey Parts, **applies_to:** All usages
- **Mechanism:** code
