# Notes

The distance between stops is specified in a SERVICE LINK . Distance and (arrival time - departure time of previous stop) can be used to calculate the speed. A range of plausible speeds for different transport modes must be defined to be able to code this test. e.g. rail 20-220 kph. coach 20 -120 kph. bus 20-90 kph. ferry 10-40 kph. air -50 -600 kph
