# Notes

The primary SERVICE JOURNEY being described with passing times etc. should reference a SERVICE PATTERN . Secondary SERVICE JOURNEYS that are declared for interchange etc. do not need to specify a pattern. Upstream numbering clash: T.03.01 is assigned to 3 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
