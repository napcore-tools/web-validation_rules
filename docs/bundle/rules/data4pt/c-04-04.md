# Notes

In some cases related elements may be declared either inline or from the root of the schema. Certain ancillary child elements such as NOTICE ASSIGNMENTs should be declared inline. Code normalized from 'c.04.04' in the source sheet. Upstream numbering clash: C.04.04 is assigned to 2 distinct rules in the source sheet; local -b/-c suffixes keep records apart.
